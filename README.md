# Lean Africa-site pregnancy-related RCTs as a transfer template

This is a GitHub-ready scaffold for the `maternal-health-e156` topic package. It contains an E156 micro-paper draft, a protocol skeleton, copied analysis outputs, and the code used to generate them.

## Included

- `paper/article.json`: E156 article record
- `paper/index.html`: self-contained HTML bundle
- `paper/e156_body.md`: frozen body plus note block
- `protocol/PROTOCOL.md`: registry-first protocol skeleton
- `data/`: copied topic-specific benchmark outputs
- `code/`: copied local scripts used for generation

## Status

- E156 body drafted and validated locally
- HTML bundle generated locally
- Protocol scaffold drafted
- Remote GitHub repo not created from this script
